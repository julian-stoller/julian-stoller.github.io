# Unused assets

"Layer-Built J" (a J sliced into 3D-print layers, amber top layer) — lives at
`/favicon.svg`. All concepts use the site palette: blue `#0366d6`, amber `#f59e0b`.

| File | Concept |
| --- | --- |
| `favicon-concept-drive-gear.svg` | Eight-tooth white gear with amber hub on blue |
| `favicon-concept-hex-head.svg` | Amber hex nut on deep navy (`#0b2843`) |
| `favicon-concept-js-plate.svg` | Heavy "JS" initials on blue (uses live text — outline the letters before production use) |
| `favicon-concept-sun-gear.svg` | Sun whose rays double as gear teeth, blue hub on white |
| `favicon-concept-j-stop.svg` | Single geometric "J" with amber full stop on white |
